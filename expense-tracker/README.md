# Expense Tracker

A simple, frontend-only Expense Tracker application built with HTML, CSS, and Vanilla JavaScript.

## Features

- **Track Finances**: Add income and expense transactions.
- **Real-time Updates**: Automatically calculates Total Balance, Income, and Expense.
- **Categorization**: Categorize transactions (Food, Travel, Shopping, Bills, Other).
- **Data Persistence**: Uses `localStorage` to save your data, so it remains available after refreshing the page.
- **Responsive Design**: Works on mobile and desktop devices.
- **Modern UI**: Clean and minimal interface with visual cues for income and expenses.

## Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Flexbox, Custom Properties (Variables), Responsive Layout.
- **JavaScript (ES6+)**: DOM manipulation, LocalStorage API, Arrow functions.

## How to Run

1. Clone or download this repository.
2. Navigate to the `expense-tracker` folder.
3. Open `index.html` in any modern web browser (Chrome, Firefox, Edge, etc.).
